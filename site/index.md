# casket-ssg

**Pure functional static site generator in Haskell.**

## Features

- Frontmatter parsing (YAML-style)
- Markdown to HTML conversion
- Mustache-style templates
- Draft post support

## Usage

```
casket-ssg build <input-dir> [output-dir]
```
